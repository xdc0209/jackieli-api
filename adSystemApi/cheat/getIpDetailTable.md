# 获取作弊Ip重复点击数## 描述
> 提供作弊详情页面作弊ip点击重复数据

## URL
> [http:// 192.168.10.168:8081/ad/cheat/getIpDetailTable.do](http://dataviewer.ilongyuan.com.cn/ad/cheat/getIpDetailTable.do)

## 支持格式
> JSON

## HTTP请求方式
> POST

## 请求参数
> |参数|必选|类型|说明|
|:-----  |:-------|:-----|-----                               |
|promoteId  |true    |String|    |
|startDate    |true    |date   |2017-03-05 |
|endDate    |true    |date   |2017-03-05|   


## 返回字段
> |返回字段|示例|说明                              |
|:-----   |:----|:-----------------------------    |
| ip    |  |ip|
|dupClickNum|  |点击重复数|
